<div id="-english">
  <h2><img src="https://i.imgur.com/gpG11L6.png" alt="English Flag" width="24"> English</h2>
  
  <p><b>WinSwift Pro</b> is an ultra-advanced system utility designed to maximize Windows performance, secure user privacy, and automate routine maintenance. Version 2.0 consolidates 9 powerful modules into a single, intelligent CLI interface that remembers your preferences and logs its actions.</p>

  ### 🚀 Key Features (v2.0)
  - **🧠 Smart Memory & Logging:** Automatically saves your language preference (EN/TR) and generates detailed operation logs in the background.
  - **🧹 Quick & Deep Cleaning:** Wipes Temp files, Prefetch, Thumbnails, Clipboard, Windows Update remnants, Event Logs, and File History.
  - **🌐 Ultimate Browser Purge:** Deep-cleans cache for Chrome, Edge, Firefox, Brave, Opera, Opera GX, and Vivaldi without losing passwords.
  - **📡 NetSweeper:** Flushes DNS, resets Winsock/TCP/IP, and reconfigures network adapters for optimal speed.
  - **🛠️ Core System Repair:** Integrated **SFC & DISM** modules to scan, repair, and restore corrupted Windows core files.
  - **🛡️ Privacy Shield:** Disables invasive Windows Telemetry, Cortana, Activity History, and background data collection.
  - **💾 Disk Optimization:** Automates TRIM/Defrag operations, clears Search Indexing, and optimizes System Restore space.
  - **📊 Hardware & System Info:** Instantly fetches and displays detailed OS, CPU, RAM, Disk, and GPU hardware information.
  - **🔧 Advanced Operations Menu:** Dedicated sub-menu for Registry optimization, Icon Cache rebuilding, Performance Monitor, and Windows Update resets.
  - **⚡ ULTRA Mode (Run All):** One-click sequential execution of all cleaning and optimization modules with post-operation statistics.

<div id="-türkçe">
  <h2><img src="https://i.imgur.com/3xLders.png" alt="Turkish Flag" width="24"> Türkçe</h2>
  
  <p><b>WinSwift Pro</b>, Windows performansını maksimize etmek, kullanıcı gizliliğini korumak ve rutin bakım işlemlerini otomatize etmek için tasarlanmış ultra gelişmiş bir sistem aracıdır. Versiyon 2.0; tercihlerinizi hatırlayan ve tüm eylemleri günlüklere (log) kaydeden, 9 güçlü modülü tek bir akıllı CLI (Komut Satırı) arayüzünde birleştirir.</p>

  ### 🚀 Temel Özellikler (v2.0)
  - **🧠 Akıllı Bellek ve Günlükleme:** Dil tercihinizi (EN/TR) otomatik olarak kaydeder ve arka planda detaylı işlem günlükleri oluşturur.
  - **🧹 Hızlı ve Derin Temizlik:** Geçici (Temp) dosyaları, Prefetch verilerini, küçük resimleri (Thumbnails), panoyu, Windows Update kalıntılarını, olay günlüklerini ve dosya geçmişini temizler.
  - **🌐 Kapsamlı Tarayıcı Temizliği:** Chrome, Edge, Firefox, Brave, Opera, Opera GX ve Vivaldi tarayıcılarının önbelleğini, şifrelerinizi kaybetmeden derinlemesine temizler.
  - **📡 NetSweeper:** DNS önbelleğini temizler, Winsock/TCP/IP protokollerini sıfırlar ve ağ bağdaştırıcılarını en iyi hız için yeniden yapılandırır.
  - **🛠️ Çekirdek Sistem Onarımı:** Bozulmuş Windows çekirdek dosyalarını taramak, onarmak ve geri yüklemek için entegre **SFC ve DISM** modüllerini içerir.
  - **🛡️ Gizlilik Kalkanı:** Müdahaleci Windows telemetrisini, Cortana'yı, etkinlik geçmişini ve arka plan veri toplama işlemlerini devre dışı bırakır.
  - **💾 Disk Optimizasyonu:** TRIM/Defrag işlemlerini otomatikleştirir, arama dizinini temizler ve sistem geri yükleme alanını optimize eder.
  - **📊 Donanım ve Sistem Bilgisi:** İşletim sistemi, CPU, RAM, Disk ve GPU donanım bilgilerini anında çeker ve görüntüler.
  - **🔧 Gelişmiş İşlemler Menüsü:** Kayıt defteri optimizasyonu, simge önbelleği (Icon Cache) oluşturma, performans izleyicisi ve Windows Update sıfırlama için özel alt menü.
  - **⚡ ULTRA Mod (Tümünü Çalıştır):** Tüm temizlik ve optimizasyon modüllerinin tek tıkla sıralı çalıştırılmasını ve işlem sonrası istatistik raporunu sağlar.
